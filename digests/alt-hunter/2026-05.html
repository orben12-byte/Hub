מסקנה ראשית

לאחר סריקה מקיפה של כל המקורות הרלוונטיים, אין כרגע חלופה שעומדת בשלושת המסננות הקריטיות.

המלצה: להמשיך עם Claude Pro ולבצע ציד נוסף ביוני 2026.

הפרויקט הכי קרוב לציפיות הוא Open WebUI Desktop — תיעוד טוב, אפליקציה native, אבל עדיין דורש agent נוסף ל-file operations.

מועמדים שנבדקו
1. Open WebUI Desktop (v0.9.5)
אפליקציית desktop מקומית לרצת AI עם Ollama — Windows / Mac / Linux
מסנן 1: file ops — נכשל מסנן 2: 6+ שלבים — נכשל מסנן 3: תיעוד — עבר
❌ נפסל — כדי לקרוא/לכתוב קבצים צריך Hermes Agent נפרד. זה 6+ שלבי התקנה = פסילה אוטומטית. אבל: אחד הפרויקטים הכי מבטיחים לעתיד אם יוסיפו file access built-in.
2. OpenWork
Open-source alternative ל-Claude Cowork, מבוסס OpenCode
מסנן 1: Windows בתשלום — נכשל מסנן 2: 6+ שלבים — נכשל מסנן 3: תיעוד — חלקי
❌ נפסל — Windows support דורש paid plan. דורש Node.js + pnpm + Rust toolchain. אין tutorial ל-Windows.
3. Kuse Cowork
Open-source, Rust-based alternative עם BYOK
מסנן 1: Docker — נכשל מיד מסנן 2: Docker — פסילה אוטומטית מסנן 3: תיעוד — חלקי (v0.0.2)
❌ נפסל — Docker = קו אדום. גם דורש API keys = לא חינמי. פרויקט מוקדם מדי (v0.0.2).
4. Gemini CLI
CLI רשמי של Google עם Gemini models
מסנן 1: Pro behind paywall — נכשל מסנן 2: 3 שלבים — עבר מסנן 3: תיעוד — עבר
❌ נפסל — אפריל 2026: Pro models הועברו מאחורי paywall. Free tier: 1,500 RPD Flash / 50 RPD Pro. לא מספיק לשימוש אינטנסיבי.
5. Claude Code + Ollama (via LiteLLM)
שימוש ב-Claude Code עצמו עם models מקומיים דרך proxy
לא חלופה — זה עדיין Claude Code
❌ לא רלוונטי — זה לא alternative, זה שימוש ב-Claude Code עם models אחרים. דורש LiteLLM proxy + 15-20 דק' setup.
עדכונים מעניינים — לא עברו, שווה לעקוב
Ollama v0.24.0-rc0 — 14 מאי 2026 תמיכה משופרת ב-vision models. Qwen 3.6 ו-Kimi K2.6 נוספו. Llama 3.2 3B הכי פופולרי.
Open WebUI v0.9.5 — 10 מאי 2026
  • Scheduled Chat Automations — משימות אוטומטיות בזמנים קבועים
  • אפליקציית Desktop native (Windows/Mac/Linux)
  • Floating chat bar (Shift+Ctrl+I)
  • Zero telemetry
MCP Ecosystem יותר מ-500 MCP servers קיימים. SDKs רשמיים: TypeScript, Python, C#, Java, Swift. אבל עדיין דורש setup מורכב מדי ל-ABC-TOM.
למה אין עדיין חלופה בשלה?
הפער המרכזי כלים קיימים נופלים בשתי קטגוריות:
  • פשוטים אבל חסרים — Open WebUI Desktop: קל להתקנה, אבל לא קורא/כותב קבצים ישירות
  • חזקים אבל מורכבים — OpenWork, Kuse Cowork: יכולים file ops, אבל Docker/6+ שלבים/API keys

מה חסר: פשוטה + חזקה + חינמית + מתועדת. שלושת אלו ביחד עדיין לא קיימים.
⚠ שינוי קריטי באפריל 2026 Gemini CLI: Pro models עברו לתשלום. זה סגר אפשרות שנראתה מבטיחה. Docker = קו אדום. Windows support עדיין בעייתי ברוב הכלים.
